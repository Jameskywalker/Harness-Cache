Agent-private L1 entries

agent_orders ptr_orders_reserve@v1:S
agent_orders_2 ptr_orders_reserve@v1:S
agent_payments ptr_payments_capture@v1:S
agent_notifications ptr_notifications_receipt@v1:S

Pointer notation

`ptr_orders_reserve@v1:S` means pointer ID, version 1, shared readable state. If the source changes, old L1 copies become `I` and must not enter prompts.

Protocol

Cache where to look

1

Agent searches pointers before broad source reads.

2

Cache opens only the verified source range.

3

Evidence is logged as opened, used, cited, or test-passed.

4

Useful pointers are promoted; stale pointers are invalidated.

Shared: readable by multiple agents after version checks.
